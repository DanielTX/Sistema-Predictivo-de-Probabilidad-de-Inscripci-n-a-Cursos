## 📋 Descripción

Sistema web predictivo capaz de identificar qué clientes tienen mayor probabilidad de inscribirse en futuros cursos de capacitación, especialmente en el área de gestión pública en Perú.

El motor predictivo funciona mediante un sistema de puntuación basado en reglas (no IA real), clasificando a los clientes en tres niveles de probabilidad: **Alta**, **Media** y **Baja**.

## 🔮 Motor Predictivo (Simulado)

| Criterio                        | Puntos |
|---------------------------------|--------|
| Participó en webinars           | +20    |
| Tiene más de 2 cursos previos   | +25    |
| Alta interacción en redes       | +15    |
| Consultas recientes             | +10    |
| Referido por otro cliente       | +10    |
| Perfil completo                 | +10    |
| Inactividad prolongada          | -15    |

| Rango     | Probabilidad |
|-----------|--------------|
| 80 – 100  | 🟢 Alta      |
| 50 – 79   | 🟡 Media     |
| 0 – 49    | 🔴 Baja      |
